# Decentralized Student Savings Wallet (DSSW)

A basic smart contract for saving ETH on the blockchain.

Users can:
- Deposit ETH (minimum 0.01 ETH)
- Withdraw after a short lock time (default 60 seconds)
- See their balance and transaction history
